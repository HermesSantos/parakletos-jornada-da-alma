# Instituto Parakletos | Landing Page

Landing page oficial do Instituto Parakletos.

Site em produção: [https://institutoparakletos.com.br/](https://institutoparakletos.com.br/)

## Tecnologias

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Como executar localmente

Pré-requisitos:
- Node.js 18+
- npm

Passos:

```bash
npm install
npm run dev
```

A aplicação ficará disponível em `http://localhost:5173`.

## Scripts disponíveis

- `npm run dev`: inicia o servidor de desenvolvimento
- `npm run build`: gera build de produção
- `npm run preview`: serve a build localmente
- `npm run lint`: executa o lint
- `npm run test`: executa os testes com Vitest

## Deploy

Este projeto possui script para deploy estático no GitHub Pages:

```bash
npm run deploy
```
